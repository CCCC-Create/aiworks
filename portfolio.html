INTERACTION DESIGN

交互设计

科技的温度，不在冰冷的数据里，而在每一次指尖触碰时界面给予的温柔回应。让交互成为一场无声的对话。

晨间日记 · App

界面设计 · iOS

一款以"晨间仪式感"为核心概念的情绪记录应用。以渐变日出色调贯穿界面，每个清晨的记录都像一次温柔的自我问候。交互逻辑遵循"最少步骤、最深情感"的设计原则。

晨起提醒 → 情绪选择 → 文字记录 → 温柔反馈

慢行 · 旅行平台

用户体验 · Web

反"快消费"的旅行体验平台，倡导"慢行即深度"。从行程规划到旅途记录，整个流程被拆解为一个个可以停顿的节点，让用户在每一步都能感受到"选择"的重量与"停留"的价值。

灵感发现 → 慢行规划 → 旅途沉浸 → 记忆收藏

呼吸 · 冥想引导

动效设计 · 可穿戴

将冥想中的呼吸节奏可视化。界面随呼吸膨胀与收缩，光圈在吸气时扩展、呼气时收拢，所有动效曲线严格匹配人体呼吸频率（4-7-8呼吸法），让界面本身就是冥想的一部分。

吸气 · 扩展 → 屏息 · 停留 → 呼气 · 收拢 → 安宁 · 状态

墨迹 · 数字书法

界面设计 · 交互逻辑

将传统书法体验移植至数字界面，笔触的粗细、墨迹的浓淡、运笔的节奏——所有物理世界的手感都被转译为细腻的触觉反馈与视觉动效。界面以宣纸肌理为底，让每一次书写都有"落笔成迹"的仪式感。

选择字体 → 触屏书写 → 墨迹生成 → 作品收藏

云南白药光钻Pro · 交互网页设计

交互设计 · 全案 · Web

为云南白药光钻Pro牙膏打造的沉浸式品牌交互网页，以「3+1钻白功效」为核心叙事线。从萌牙IP角色到剧情式分镜引导，从滚动触发动画到点击微交互反馈，将产品功效转化为可感知的数字体验。涵盖平面视觉、交互流程、UI界面、用户动线与空间版式的完整设计体系。

剧情引入 → 功效展示 → 互动体验 → 科普对比

ABOUT & RESUME

关于我

设计是温柔的抵抗——抵抗平庸、抵抗遗忘、抵抗那些让世界变得粗糙的力量。

蔡辰

跨领域设计师 · 平面 / 空间 / 交互

我相信好的设计不是被看见的，而是被感受到的。它不需要喧哗，只需要安静地存在，然后改变你看待世界的方式。

我的创作始终围绕一个核心追问：如何在理性与感性之间找到那个恰到好处的平衡点？平面设计让我理解秩序与美感的关系，空间设计教会我感知光影与尺度的语言，交互设计则让我相信——科技最动人的瞬间，是它变得像一首诗那样安静的时候。

我追求的不是风格的统一，而是气质的连贯——温柔、克制、有呼吸感，像晨雾中的远山，而非烈日下的棱角。
